```cmd
  nvm arch                     : 当时是32/64位.
  nvm current                  : 当前使用版本.
  nvm install <version> [arch] : 安装node, "latest" 代表最新
  nvm list [available]         : 别名为ls, 加上available代表当前可安装版本
  nvm on                       : 启用 node.js 管理.
  nvm off                      : 禁用.
  nvm proxy [url]              : 设置代理.
  nvm node_mirror [url]        : 设置node 镜像
  nvm npm_mirror [url]         : 设置npm 镜像
  nvm uninstall <version>      : 卸载
  nvm use [version] [arch]     : 使用
  nvm root [path]              : nvm目录
  nvm [--]version              : -v,当前nvm版本-v

https://www.jianshu.com/p/de9d32967f49
```
