扎克·洛瓦特的 Scriptlets

一系列用于kbar2、工具启动器、快速菜单 2或任何其他脚本文件启动器的小型、简单的 AE 实用程序。

请注意，这些都在开发时起作用；随着 AE 的发展，有些可能会停止工作。如果出现任何故障，我绝对不承担任何责任；自行决定运行💝

如果这些真的在某种程度上帮助了你，请随意扔一点咖啡吧！

所有工具 © 2021 Zack Lovatt。版权所有。

安装说明

  • 单击您要下载的脚本的名称
  • File > Save 这到您的计算机，确保您保存为“[文件名].jsx”
  • 在AE中：File > Script > Run Script File...并选择这个脚本！

Adobe Illustrator 脚本

有关一些 AI 脚本，请参阅Illustrator文件夹。

添加随机效果

将随机图层效果添加到您的第一个选定图层 🙃

添加修剪路径

将修剪路径添加到选定的形状层，包括一个开始的关键帧和一个结束动画的关键帧。

修饰符 影响
转移 跳过添加关键帧

添加摆动路径

将摆动路径添加到选定的形状图层。

打开脚本并更改摆动大小、详细信息和频率数字以在运行脚本时自动设置这些数字。

将颜色文件应用到 Colorista Free

提示用户选择 CCC/CDL，使用来自 CDL 的设置创建带有 Colorista Free 的调整层。

闪烁所选图层

通过将每个图层的不透明度链接到中央复选框，添加场景闪烁控制器并闪烁所选图层。

当复选框关闭时，这些层关闭。当复选框为 ON 时，图层也是如此。

修饰符 影响
转移 反转闪烁表达式（复选框关闭时图层打开）

将新的高斯模糊转换为旧的

将新的 (CC 2015.3+) 高斯模糊效果转换为高斯模糊（旧版）

计算关键帧

计算项目面板中选定合成中的所有关键帧（包括标记），如果未选择则计算打开的合成中的所有关键帧（包括标记）。将递归到预压缩。

创建平均位置空值

选择一些图层；将在它们的平均位置创建一个新的空值。

修饰符 影响
转移 将位置设置为当前时间的固定值，而不是使用动态表达式

默认文件夹 v2.0

创建多个文件夹（或者，如果作为窗口运行，则允许您自定义要创建的文件夹名称）。

基于Impudent1 的 DefaultFolders。

及时删除键

删除当前时间的所有键

禁用所有 Comp 表达式

禁用活动组合中的所有表达式

冻结属性

使用表达式将选定的属性（或图层）冻结为当前值。

再次运行脚本以解冻选定的属性/图层。

笔记：
  • 跳过已有表达式的属性。

生成字体粗细选择器

查看现有文本图层，并尝试根据现有文本图层的字体创建表达式以选择字体粗细。

使用：
  • 选择一个文本层
  • 运行脚本
  • 它将在文本层上创建一个滑块，并在 sourceText 上创建一个表达式
  • 滑块将从 1 到找到的权重数量进行动画处理
  • 该表达式将列出它可以检测到的所有可用字体权重！
笔记：
  • 这可能不适用于所有情况。字体很奇怪。
  • 这依赖于 CC 2019+。
  • 来自10 的字体 PostScript 名称逻辑并在 MIT 下获得许可（包含在文件中）

获取属性值

显示所选属性的当前值（在表达式之后），让您可以将其复制并粘贴到其他地方。

如果作为脚本（或从 kbar）运行，面板将立即显示该值。

如果作为 ScriptUI 面板运行（如果安装在 AE 中），您可以停靠面板以快速获取值。

笔记：
  • 该面板还包括一个实验性的“复制到剪贴板”功能；实际复制到剪贴板通常需要几秒钟的时间，因此请耐心等待。

保留每 N 个键

查看您的关键帧并保留其中的每一个。

默认情况下，将保留每个第二个关键帧，并删除其余的。请注意，始终保留第一个和最后一个关键帧。

用于烘焙表达式，例如使用 Bodymovin 导出时。

修饰符 影响
转移 指定间隔值

密钥集

保存和加载选定的关键帧集，即使时间或值发生变化。

您可以通过保存现有密钥集来覆盖选择。

顶部按钮将保存选择，而底部按钮将加载该选择。

修饰符 影响
转移 指定集合的​​名称
如果作为 kbar 脚本按钮运行：
  • 指定一个唯一的 Kbar 参数来保存/恢复该密钥集
  • 你可以有无数个集合，只要每个集合都有一个唯一的名字
  • 按下按钮保存
  • 按住 SHIFT 加载该选择。
笔记：

  • 如果添加中间关键帧、删除图层或属性、添加额外效果等，Key Sets 可能无法恢复完整选择。

关键绊脚石

获取一对关键帧，并在它们之间添加额外的随机绊倒、交错的关键帧。

有助于制作逼真的进度条，除此之外别无他法！

要创建的关键帧数量的选项，每个关键帧被保持或贝塞尔曲线的机会（相对于线性，默认值）；bezier 关键帧自动或连续的机会，以及关键帧之间保留的最小帧数。

加载上次自动保存

尝试根据您当前的 AEP 查找最后一个自动保存项目，并将其打开。

笔记：
  • 目前仅适用于 AE 的英文安装。
